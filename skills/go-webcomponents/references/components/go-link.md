# Link Component

The `go-link` component creates a link to the predefined destination.

## Example
```html
<go-link to="checkoutform">Checkout</go-if>
```

## Properties

| Property | Attribute | Description     | Type   | Default |
|----------|-----------|-----------------|:-------|:--------|
| `to`     | `to`      | The destination | String | ``      |

### to
Is the name of a predefined destination that clicking on this link will lead to.
This list is defined via go config.
