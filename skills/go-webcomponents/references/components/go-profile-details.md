# Profile Details Component

The `go-profile-details` component allows authenticated users to view and edit their account details.

## Example

```html
<go-profile-details></go-profile-details>
```

## Form Fields

The component includes the following fields:

- `salutation` (optional) - User's salutation/title
- `firstName` (required) - User's first name
- `lastName` (required) - User's last name
- `email` (required) - User's email address
- `confirmEmail` (required) - Email confirmation field
- `language` (required) - User's preferred language
