# Sign Up Component

The `go-sign-up` form to register users to a shop.

## Interactive Example

Edit the code below and click **Run** to see live changes in the preview pane.

## Example

```html
<go-sign-up></go-sign-up>
```

## Registration Fields

### Mandatory

This fields are mandatory for a customer to be able to register on the backend

- `First Name`
- `Last Name`
- `Email`
- `Email Confirmation`
- `Password`
- `Password confirmation`
- `Terms and Conditions`

### Optional

- `Salutation`
- `Addressee`
- `Street`
- `Postal code`
- `Country`
- `Language` (when not provided will default to backend default language)
